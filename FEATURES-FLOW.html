Resumo de Todas as Features
FEATURE 01
⏱️ SLA / Tempo de Resposta
Indicador visual de urgência nos cards. Borda vermelha se sem atualização há +24h, amarela se +12h.
Card.tsx Card.module.css ✅ Concluída
FEATURE 02
🔍 Filtros Avançados
Filtragem por prioridade, responsável e etiqueta com dropdowns na barra do header e badge de contagem ativa.
KanbanBoard.tsx ✅ Concluída
FEATURE 03
⌨️ Atalhos de Teclado
Hook customizado com 8 atalhos: N (novo), F (filtros), / (busca), Ctrl+K (busca), R (refresh), C (compacto), ? (ajuda), Esc (fechar).
useKeyboardShortcuts.ts KanbanBoard.tsx ✅ Concluída
FEATURE 04
📋 Templates de Ticket
Salvar, carregar e excluir templates no modal de Novo Ticket. Persiste no localStorage.
KanbanBoard.tsx localStorage ✅ Concluída
FEATURE 05
📊 Métricas no Dashboard
6 cards: total, abertos, resolvidos, alta prioridade, criados hoje e taxa de produtividade.
DashboardView.tsx ✅ Concluída
FEATURE 06
📥 Exportação CSV
Botão de download CSV com BOM para compatibilidade Excel. Inclui todos os tickets ativos com data.
DashboardView.tsx ✅ Concluída
FEATURE 07
💬 Integração WhatsApp
UI de configuração pronta (InstanceModal.tsx). Falta Supabase Edge Function para comunicar com API chatPro.
InstanceModal.tsx ⏳ Pendente Requer backend
FEATURE 08
🔔 Notificações Push Nativas
Browser Notification API. Solicita permissão no login. Dispara notificação nativa quando aba não está focada.
NotificationContext.tsx ✅ Concluída
FEATURE 09
🔒 Notas Internas
Toggle Público/Interno na caixa de comentário. Notas internas com fundo âmbar, badge "NOTA INTERNA" e ícone de cadeado.
CardDetailModal.tsx ✅ Concluída
FEATURE 10
☑️ Seleção Múltipla
Modo bulk: seleciona vários cards, barra flutuante com ações de mover em lote e arquivar em lote.
KanbanBoard.tsx Card.tsx ✅ Concluída
FEATURE 11
📐 Modo Compacto
Cards renderizados em linha única: ponto de prioridade + título (ellipsis) + avatar. Atalho: tecla C.
Card.tsx Card.module.css KanbanBoard.tsx ✅ Concluída
FEATURE 12
🎨 Temas por Usuário
8 presets de cor (Escuro, Profundo, Claro, Verde, chatPro, Roxo, Sunset, Oceano) + personalização. Persiste no localStorage.
theme.tsx KanbanBoard.tsx ✅ Concluída
FEATURE 13
🤖 Regras Automáticas
Criar regras condição→ação no Settings. Condições: prioridade, sem assignee, overdue. Processa auto ao carregar o board.
KanbanBoard.tsx localStorage ✅ Concluída
FEATURE 14
🔗 Webhook de Entrada
Endpoint para receber chamadas externas e criar tickets automaticamente. Precisa de Supabase Edge Function.
⏳ Pendente Requer backend
