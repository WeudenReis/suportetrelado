10. Features Implementadas
Concluída
Pendente (requer backend)
1. SLA / Tempo de Resposta
Borda visual vermelha (24h+) ou amarela (12h+) nos cards. Card.tsx + Card.module.css
2. Filtros Avançados
Dropdowns por prioridade, responsável e etiqueta na barra do header. KanbanBoard.tsx
3. Atalhos de Teclado
Hook customizado com N, F, /, Ctrl+K, R, C, ?, Esc. useKeyboardShortcuts.ts
4. Templates de Ticket
Salvar/carregar/excluir templates no modal Novo Ticket. localStorage.
5. Métricas no Dashboard
6 cards de métricas: total, abertos, resolvidos, alta prioridade, criados hoje, produtividade.
6. Exportação CSV
Botão de download CSV com BOM para Excel. DashboardView.tsx
7. Integração WhatsApp
InstanceModal.tsx já tem campos. Precisa de Edge Function para API chatPro.
8. Notificações Push Nativas
Browser Notification API quando aba não está focada. NotificationContext.tsx
9. Notas Internas
Toggle Público/Interno no comentário. Visual âmbar com ícone 🔒. CardDetailModal.tsx
10. Seleção Múltipla (Bulk)
Selecionar vários cards, mover em lote ou arquivar em lote. Barra flutuante.
11. Modo Compacto
Cards em linha única com ponto de prioridade, título e avatar. Tecla C.
12. Temas por Usuário
8 presets (Escuro, Profundo, Claro, Verde, chatPro, Roxo, Sunset, Oceano) + customizado.
13. Regras Automáticas
Condições (prioridade, sem assignee, overdue) → ação de auto-mover. Settings panel.
14. Webhook de Entrada
Precisa de Supabase Edge Function para receber chamadas externas e criar tickets.
